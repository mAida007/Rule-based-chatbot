Beginner-Level Chatbot for Software Business (Python Only)
==========================================================

Description:
------------
This is a rule-based chatbot built using Python. It answers frequently asked questions related to software services.

How to Run:
-----------
1. Make sure you have Python installed.
2. Open a terminal or command prompt.
3. Navigate to the folder where this chatbot is located.
4. Run the bot with:
   python chatbot.py

Files Included:
---------------
- chatbot.py : The main Python file for the chatbot.
- faq.json    : Sample dataset with predefined FAQs.
- README.txt  : This instruction file.

Notes:
------
- This bot uses keyword matching and is intended for beginner-level projects.
- It does not collect or store user data.
